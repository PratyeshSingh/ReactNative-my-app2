# SDK and Library Details: Product Browser App

## 1. Core Dependencies

| Dependency | Version | Purpose |
| :--- | :--- | :--- |
| `expo` | `~54.0.33` | Core framework for universal React applications. Provides access to native APIs and managed workflow. |
| `expo-router` | `~6.0.23` | File-based routing system for React Native, enabling seamless navigation similar to Next.js. |
| `react-native` | `^0.81.5` | The core mobile UI framework used for rendering native components. |
| `react` | `^19.1.0` | The primary UI library for building component-based interfaces. |

## 2. UI & Navigation Libraries

| Dependency | Version | Purpose |
| :--- | :--- | :--- |
| `expo-image` | `~3.0.11` | Optimized image component with support for caching, placeholders, and smooth transitions. |
| `@react-navigation/native` | `^7.3.13` | Underlying navigation primitives used by Expo Router. |
| `react-native-reanimated` | `~4.1.1` | Powerful animation library for high-performance fluid interactions. |
| `expo-status-bar` | `~3.0.9` | Manages the device status bar appearance. |

## 3. Development & Testing Tools

| Dependency | Version | Purpose |
| :--- | :--- | :--- |
| `jest` | `^29.7.0` | JavaScript testing framework for unit and integration tests. |
| `ts-jest` | `^29.1.0` | TypeScript preprocessor for Jest, allowing tests to be written in TS. |
| `typescript` | `~5.9.2` | Static typing for JavaScript, ensuring code quality and better developer experience. |
| `eslint` | `^9.25.0` | Pluggable linting utility for identifying and reporting patterns in JS/TS. |

## 4. API & Data Handling
- **Fetch API**: Standard built-in browser/mobile API used for networking (Product API calls).
- **TypeScript Interfaces**: Used for defining DTOs (Data Transfer Objects) and Domain Entities for type-safe data handling.
